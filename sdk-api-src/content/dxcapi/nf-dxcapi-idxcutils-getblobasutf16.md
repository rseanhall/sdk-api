## -description

Convert or return matching encoded text blob as UTF-16.

**GetBlobAsUtf16** is an alias to **GetBlobAsWide** on Win32.

## -parameters

### -param pBlob

The blob to convert.

### -param pBlobEncoding

The address of the pointer that receives a pointer to the newly-created blob.
